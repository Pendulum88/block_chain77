React
 - React란 FaceBook에서 만든 라이브러리이다
 - View만을 위해 제작되었다 << npm/yarn을 이용하지만 BackEnd가 아니다
 - yarn이라는 패키지 매니저를 사용한다 (npm으로 안된다는 이야기는 아니다) 
 - Javascript와 HTML을 함께 js파일에 작성한다 << 원래는 JSX파일에 작성해야한다 (*.jsx)
    (*.jsx) (tsx, ts는 정확하게 지켜져야한다. 단 js, jsx는 크게 따지지 않는다)
    - JSX : Javascript XML의 약자로 Javascript를 확장한 문법이다
    - React의 대표적인 특징 중 하나
 - Component는 구조의 단위이다. HTML, Javascript를 포함한다
 - React 상에서만 사용하는 DOM이 있다. 렌더링에 바로 적용되진 않는다
    - 해당 DOM을 Virtual DOM(가상 돔) 이라고 한다
    - React는 Virtual DOM을 수정하고 수정된 부분만을 실제 DOM에 적용한다
    - let, var 등등은 변화하더라도 React에서 감지하지 못한다
    - 변화를 감지했을때만 그 부분만을 리랜더링(다시그리기) 하기 때문에 let 을 사용하는 경우 변화가 없다
 - state라는 단위가 있다
    - state : 상태 즉, 현재 React의 상태를 확인하고 변경 되었을때 그 변경 사항을 React에 알려준다
    - 즉, 변화시 리랜더링이 발생한다
