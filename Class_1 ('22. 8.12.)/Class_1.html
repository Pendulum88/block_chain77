HTML 기본구성


신규 HTML파일 생성 후 DOC 엔터를 하면 HTML 구성 기본 틀이 쓰여진다
html : html 문서의 범위를 상징 (참고 : 페이지이동)
head : html 문서의 기본정보를 설정 (참고 : 페이지이동)
body : html 문서의 본문 요소 (참고 : 페이지이동)

meta : 메타데이터, 간단히 말해서 기본정보를 전달한다
charset : 언어가 어떠한 포멧(설정)인지 설정. UTF-8은 유니코드의 일종인데, 자바스크립트할때 자세히 다룰 예정
http-eqiv : 호환성에 대한 부분(어떤 호환성을 가지고 있는지? 자바스크립트를 어디까지 사용할수 있는지?)
content : 브라우저가 어디까지 패치가 되었냐에 대한 내용
viewport : 브라우저가 어떤크기를 갖고있는지, 어디에 맞추면되냐에 대한 설정
content : 길이를 어느 기준으로 할것인지 설정
initial-scale : 기본확대를 어떻게 할거냐 스케일 조정에 대한 내용 (확대축소)
title : 사이트 이름
