내용이 너비를 넘으면 줄바꿈을 한다. 브라우저가 자동으로 위쪽과 아래쪽에 약간의 margin을 추가한다. 이렇게 태그안에 다른 엘리먼트를 넣을수도 있다

span태그, p태그와 유사, 인라인속성. 자동으로 줄바꿈이 되지 않는다. 
        pre태그, preformatted, 블록속성.

style : 속성, 태그의 크기, 외곽선, 위치, 배경 등등을 설정 가능하다.

너비나 늘어난 숫자를 이용한 수치들에 대한 단위 : px, %, em, rem, vw, vh

px : 픽셀 그냥 한칸 한칸 % : 부모를 기준으로한 퍼센테이지, 100% = 부모의 너비와 같다

em : 상위 요소 기준
rem : 최상위 요소(html) 기준

1rem은 (일반적으로)16픽셀이다.

vw, vh : viewport, 브라우저의 크기나 스마트폰의 크기를 기준으로 %, 100vw = 화면의 너비, 100vh = 화면의 높이

width : 영역의 너비 max-width : 영역의 최대 너비 min-width : 영역의 최소 너비 %, vw, vh는 기준이 있고 그 기준의 비율로 너비를 설정하지 어떤 내용을 넣었는데, 최소 100px은 있어야 제대로 뜬다? min-width 어떤 내용을 넣었는데, 너무 크면 안돼, 1000px 이상 넘어가면 안된다, 그러면 max-width
height : 높이 max-height : 영역의 최대 높이 min-height : 영역의 최소 높이 보통 이미지 조절할때 아니면 쓸일이 많이 없다
